## Installation

```sh
cd
git clone https://github.com/cbandy/vim-config.git .vim
ln -s .vim/vimrc .vimrc

.vim/lua/plugins.lua +PlugUpgrade +PlugInstall
```

## Update

```sh
~/.vim/lua/plugins.lua +PlugClean +PlugUpdate
```
