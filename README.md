## Installation

```shell
cd
git clone https://github.com/cbandy/vim-config.git .vim
ln -s .vim/vimrc .vimrc
ln -s .vim/gvimrc .gvimrc
vim -u .vim/plugins.vim '+PlugUpgrade' '+PlugInstall'
```
